# Changelog

## [1.0.0] - 2025-01-01

### Initial Release
- Slipstream DNS tunnel layer for 3x-ui / V2Ray panels
- Auto-parse V2Ray / VLESS / VMess / Trojan links
- Selectable tunnel port (53, 80, 443, 8080, or any custom port)
- iptables redirect with systemd persistence
- Auto-detection of server IP
- `picostream` CLI for management (start/stop/restart/logs/uninstall)
- Support: Ubuntu, Debian, CentOS, Rocky Linux
- Architecture: amd64, arm64, armv7
